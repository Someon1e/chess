## Generic chess engine written in rust

### Features:
- UCI
- Legal move generator
- Alpha-beta pruning
- Move ordering
- Killer move heuristic
- Late move reduction
- Transposition table
- Quiescence search
- Iterative deepening
- PeSTO evaluation
- Stalemate and checkmate detection
- Repetition detection
